## 📝 로드맵 핵심 순서

**Phase 1: 골격** — 프로젝트 초기 설정 (패키지 설치, 타입 정의, 환경변수 스키마, UUID 미들웨어)

**Phase 2: 공통** — Invoice UI 컴포넌트 (InvoiceHeader, ItemTable, Summary, Meta, not-found, 반응형)

**Phase 3: 핵심 기능** — Notion API 연동 및 견적서 뷰어 (getInvoiceBySlug, GET API, 뷰어 페이지)

**Phase 4: 추가 기능** — PDF/인쇄, 슬러그 재발급 API, 캐시 무효화 API, 다크모드

**Phase 5: 최적화/배포** — Notion 샘플 데이터, 환경변수 설정, Vercel 배포, E2E 테스트, Lighthouse 성능 측정
